# The Sack

A web app for learning the 0/1 Knapsack Problem — visualize the DP solution and play a packing game.

**[Try it live](https://the-sack-game.web.app)**

## Tabs

- **Problem** — explains the knapsack problem, why greedy fails, and how DP solves it
- **Solution** — generates random items, builds the DP table, highlights the optimal path
- **Game** — 10 levels, pick items to maximize value, watch them drop into a bag with 2D physics

## Run locally

Open `index.html` in a browser.

## Tech

HTML/JS (no build step), Tailwind CSS (CDN), Matter.js (physics), poly-decomp (colliders).
